# Tutorial Vue3-COMPOSITION-API ON the SERVER SIDE
This tutorial aims to illustrate the use of vue3 with COMPOSITION-API on the server side served by @vue/cli-service

# Install dependencies
#initiate wb-ui-3 project

npm init
#install local node_modules

npm i
#install last version of vue (3.2.26)

npm install vue@next
At this point, we cam work at node shell

# Creating server
#Install service provided by @vue framework (we can use express framework)

npm install vue-cli-service
# add services from @vue-cli-service in packages.json->scripts

scripts:{... "serve": "vue-cli-service serve", }
